# OpenClaw Mastery: 1-Day Intensive Workshop

> Status: DRAFT | Last updated: 2026-03-20

## Workshop Title Options

- OpenClaw Mastery: Deploy Your Organisation's AI Agent in One Day
- The Agentic Enterprise: Hands-On OpenClaw for Leaders
- [Refine once positioning is locked]

## Target Audience

C-suite and board-level leaders. Smart, busy, non-technical but not afraid of technology. They want to understand what this thing is, why it matters, and how to deploy it safely.

## Format

- 1 day, in-person
- 10-30 participants (ideal)
- Hands-on, practical, live
- Every participant leaves with a working OpenClaw deployment
- Consistent with Justin's existing workshop methodology

## Proposed Structure (8 hours)

### Module 1: Why Agentic AI Changes Everything (60 min)
- The shift from reactive to autonomous AI
- OpenClaw in context: what it is, where it came from, why it matters
- Live demo: show what a well-configured OpenClaw can do
- The business case for autonomous agents

### Module 2: Architecture and Core Concepts (45 min)
- How OpenClaw works under the hood (gateway, workspace, agents)
- The config stack: AGENTS.md, SOUL.md, TOOLS.md, HEARTBEAT.md
- Messaging platform integrations
- Models and API costs

### Module 3: Security First (60 min)
- The threat landscape (Gartner, Cisco, real incidents)
- Prompt injection, data exfiltration, supply chain attacks
- Hardening your deployment
- What your board needs to know about risk
- **This module is non-negotiable for this audience**

### Module 4: Installation and First Agent (90 min)
- Guided setup: install OpenClaw, configure workspace
- Write your first AGENTS.md and SOUL.md
- Connect to a messaging platform
- Send your first message and get a response
- **Hands-on exercise: Every participant has a working agent**

### LUNCH (60 min)

### Module 5: Heartbeat and Autonomous Operation (60 min)
- Configure heartbeat for scheduled tasks
- Write a HEARTBEAT.md checklist
- Set active hours and monitoring
- Cost management
- **Hands-on exercise: Set up a heartbeat routine**

### Module 6: Skills and Capabilities (60 min)
- The skills ecosystem
- Safe skill sourcing (ClawHub risks)
- Writing custom skills
- Skill filtering and trust levels
- **Hands-on exercise: Install and configure skills**

### Module 7: Advanced Patterns (45 min)
- Multi-agent setups
- Agent specialisation and orchestration
- Integration with other AI tools (Claude Code, Copilot, etc.)
- Scaling across teams

### Module 8: Your 90-Day Deployment Plan (45 min)
- Days 1-30: Quick wins and safe expansion
- Days 31-60: Scaling to team use
- Days 61-90: Embedding into operations
- Governance framework
- **Exercise: Each participant writes their 90-day plan**

### Wrap-Up and Q&A (15 min)

## Participant Prerequisites

- Laptop with Node 22+ installed (pre-workshop setup guide)
- API key for at least one supported LLM
- A messaging platform account for testing (WhatsApp, Slack, or Telegram)
- Basic comfort with a terminal (we'll guide them, but they need to be willing)

## Deliverables for Participants

- Working OpenClaw deployment on their machine
- Reference configuration (security-hardened)
- 90-day deployment plan (personalised)
- Cheat sheet / quick reference card
- Post-workshop support channel access
