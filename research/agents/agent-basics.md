# OpenClaw Agents

> Last updated: 2026-03-20

## How Agents Work

- Config-first definition. Write an `AGENTS.md` file, run a command, agent is live.
- Multiple agents can be defined with separate workspaces, histories, and tool access.
- Bindings map specific conversations to specific agents.
