# OpenClaw Overview

> Last updated: 2026-03-20 | Version: v2026.3.13-1

## What Is OpenClaw

Free, open-source autonomous AI agent. Runs locally. Connects LLMs to real software through messaging platforms (WhatsApp, Telegram, Slack, Signal, Discord).

Not a coding tool. A general-purpose life and work automation agent that happens to be able to execute code.

## Origin

- Created by **Peter Steinberger** (founder of PSPDFKit, 13 years).
- Launched November 2025 as "Clawdbot."
- Renamed "Moltbot" (Jan 27, 2026) after Anthropic trademark complaints.
- Renamed **"OpenClaw"** three days later.
- Feb 14, 2026: Steinberger joined OpenAI. Project moved to independent open-source foundation (MIT licence). OpenAI is sponsor.

## Scale

- 250,000+ GitHub stars in under 4 months.
- Estimated 300,000 to 400,000 users.
- Jensen Huang (Nvidia): "Probably the single most important release of software... probably ever."
- Multiple releases per week. Very active development.

## What It Can Do

- Read/write files
- Run shell commands
- Browse websites
- Send emails
- Control APIs
- Schedule meetings
- Execute code
- Book flights
- 24/7 background operation via heartbeat

## Practical Positioning

Use OpenClaw for: browser automation, reminders, email, scheduling, 24/7 assistant tasks.
Use Claude Code for: heavy coding, codebase work.
Use Codex for: low-code agentic workflows.

They complement each other. They don't compete.
